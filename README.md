# Communication System Simulation

Two complete digital communication system simulations with error correction coding.

## Project 1: Basic QPSK System
- QPSK modulation with Hamming (7,4) error correction
- AWGN channel simulation
- BER calculation and constellation plots

## Project 2: Advanced 16-QAM System  
- 16-QAM modulation with Hamming (7,4) code
- Transmit/Receive filters with upsampling
- GEO satellite path loss model
- BER vs SNR curve generation
